# LifePath 
**Venez chômeur, repartez travailleur**

LifePath est une plateforme web dédiée aux **étudiants et jeunes diplômés** à la recherche de **stages** et **contrats en alternance**.  
L’objectif est de simplifier l’accès à l’emploi, de l’orientation à la candidature, dans un environnement **sécurisé**, **moderne** et **accessible**.

---

##  Fonctionnalités principales

- 🔍 Recherche d’offres de **stage** et **alternance**
- 🧑‍🎓 Espaces dédiés **Étudiants**
- 🏢 Espaces dédiés **Entreprises / Recruteurs**
- 📄 Dépôt et gestion de CV
- 📨 Candidature en ligne
- 🔐 Authentification sécurisée
- 📊 Tableau de bord personnalisé
- 📱 Interface responsive (mobile / tablette / desktop)

### Typographies

- **Titres** : Police sans-serif moderne et lisible (ex : *Poppins*, *Montserrat*, *Inter*)

- Titres : **gras / impact visuel**
- Contenus : **clairs et aérés**

## Sécurité & Hébergement

Le site est conçu pour être **hébergé sur un serveur sécurisé**, avec :

- 🔒 Certificat SSL (HTTPS)
- 🔑 Gestion des rôles (admin / étudiant / recruteur)
- 🛡️ Protection contre les injections SQL et XSS
- 🔄 Sauvegardes régulières
- 📁 Sécurisation des fichiers uploadés (CV, documents)

- **Frontend** : HTML, CSS, JavaScript (ou React / Vue)

- **Serveur** : Linux (Apache / Nginx)
